下单&支付

  2. 2. 浏览心仪商品，下单
  3. 3. 选择合适支付方式

    2. 3.2 隔日付/当面付。可以通过支付宝/pos机/现金方式，最长支付期限达5天。
  4. 4. 支付成功将有回单，请向工作人员索取。(在线支付回单会随商品送达。)

配送说明

  1. 1. 时间: 初定每日6：30AM-10:30AM
  2. 2. 地点: 小区直配--送至小区固定位置(详见小区目录)
  3. 3. 分发: 除老弱病残等不便自取者外，所有货物请客户在上述时间范围内至上述地点自提。
  4. 4. 逾期未收货: 顺延至次日分发。(生鲜订单除外)，建议与邻居协商代领。

收货&确认

  1. 1. 随货配有订单，共三联(订单联、支付确认联及送货联)
  2. 2. 订单联客户收存
  3. 3. 支付确认联，如已支付，由客户保存，否则签字后交至工作人员处。
  4. 4. 送货联，客户签字后交由工作人员收回。
